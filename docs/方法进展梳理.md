# EEG_TNP 工作进展梳理

## 背景

EEG 领域中的对抗防御目前仍然主要由对抗训练类方法主导，例如 Madry 对抗训练、TRADES、ABAT 等。这类方法虽然能够在一定程度上提升模型对特定攻击的鲁棒性，但也存在几个明显问题：

一方面，对抗训练通常需要在训练阶段绑定特定的攻击方式、扰动强度和分类器结构，导致防御策略的灵活性不足；另一方面，EEG 信号本身具有低信噪比、非平稳性和个体差异强等特点，判别性特征往往较弱，因此单纯依赖分类器层面的鲁棒优化容易造成过拟合，并引发 clean accuracy 与 robust accuracy 之间的权衡问题。

此外，对抗训练主要作用于模型参数，而不是直接修复被扰动的输入信号，因此在面对未知攻击或跨模型场景时泛化能力仍然有限。

## 动机

为了进一步提升 EEG 对抗防御的灵活性、鲁棒性和可部署性，本工作希望从传统的分类器中心防御范式转向输入级别的测试时净化范式。

相比于仅通过对抗训练增强下游分类器，输入净化方法可以在样本送入分类器之前先削弱对抗扰动，从而减少分类器对特定攻击方式和训练策略的依赖。

考虑到 EEG 信号在时空维度上具有一定的结构性和潜在低秩特征，而对抗扰动往往表现出更不规则、更高频、低秩性更弱的特征，本工作引入张量网络建模 EEG 信号，通过低秩张量重构实现对抗扰动抑制和有效神经语义保留。

## 方法

本工作提出了一种基于渐进式张量环的 EEG 对抗净化策略。具体而言，输入 EEG 样本首先被转换为张量表示，然后通过 Progressive Tensor Ring（PTR）进行由粗到细的低秩重构。

PTR 在低秩约束下对 EEG 信号进行近似重建，使模型优先恢复信号中的主导结构和任务相关成分，同时抑制对抗扰动中不规则、高频和低秩性较弱的部分。

样本经过 PTR 净化后，再被送入下游分类器进行识别，从而形成一种输入级、测试时可部署的对抗防御流程。

为了缓解张量网络净化可能带来的分布漂移问题，本工作进一步在下游分类器的对抗训练过程中引入少量净化样本。具体来说，在训练分类器时，不仅使用干净样本和对抗样本，还将经过 PTR 净化后的样本加入训练集，使分类器提前适应净化样本的输入分布。

同时，在 logits 层面引入一致性约束，将干净样本的预测 logits 作为 teacher，将对应净化样本的预测 logits 作为 student，通过二者对齐来减少净化过程造成的语义偏移和分类不稳定。

该策略能够提升分类器对净化样本的兼容性，降低由于输入分布变化导致的性能波动。

此外，PTR 的净化强度与 rank 参数高度相关。较低 rank 往往具有更强的压缩和净化能力，但可能损伤 EEG 中细粒度的判别信息；较高 rank 可以更好地保留原始信号细节，但也可能同时恢复部分对抗扰动。

因此，固定 rank 难以适应不同 EEG 样本之间的差异，限制了净化策略的灵活性。针对这一问题，本工作进一步引入基于分类结果稳定性和信号保真性的自适应 rank 选择机制。

在渐进式 rank 增大的过程中，该机制综合判断分类结果是否趋于稳定，以及重构信号是否仍保持足够的保真度，从而为不同样本选择更合适的净化强度，实现更加灵活和样本自适应的 EEG 对抗净化。

## 实验

目前实验主要在 SSVEP 数据集和 EEGNet 分类器上进行验证。

实验中将 EEG_TNP 与 Madry 对抗训练、TRADES、ABAT 等代表性 EEG 对抗防御方法进行对比，并在不同攻击设置下评估 clean accuracy 和 robust accuracy。

现有结果表明，基于 PTR 的低秩净化能够有效削弱 EEG 对抗扰动，净化样本感知的对抗训练能够提升下游分类器对净化输入的适应能力，而自适应 rank 选择机制进一步改善了净化强度固定所带来的 clean accuracy 与 robust accuracy 权衡问题。

整体来看，EEG_TNP 在当前 SSVEP 数据集和 EEGNet 设置下取得了 SOTA 防御性能。
